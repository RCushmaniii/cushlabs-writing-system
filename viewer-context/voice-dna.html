About This File

When to load: Every writing task

The Voice DNA file captures the distinctive patterns that make content sound like the founder, not a generic AI. It includes:

  • Core tone and attitude
  • Sentence rhythm and structure preferences
  • Distinctive phrases and words to favor
  • Explicit boundaries (what never to say)
  • Punctuation and formatting style
