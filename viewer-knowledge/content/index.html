How to Use This Content

When creating new assets:

  1. Check if a gold-standard example exists for the content type
  2. Review the example to understand tone, structure, and phrasing patterns
  3. Adapt (don't copy verbatim) to fit the new context
  4. Ensure all Context files (Voice DNA, ICP, Business Profile, Claims Policy) are loaded

Gold-standard content serves as a reference, not a template to be copied wholesale.
