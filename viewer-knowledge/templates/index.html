What Templates Are

Templates are pre-written, approved blocks that can be inserted into content with little or no modification. They include:

  • CTAs — Calls to action with proven conversion language
  • Proof blocks — Social proof, testimonials, credentials
  • Bios — Founder/company descriptions at various lengths
  • Objection handling — Pre-mapped responses to common blockers
  • Snippets — Microcopy, taglines, value props

Templates are reusable as-is unless adaptation is explicitly required for context.

How to Use Templates

When creating new content:

  1. Check if a template exists for the component you need (CTA, proof, bio, etc.)
  2. Use the template verbatim unless context requires adaptation
  3. Ensure the template aligns with current Context files (offers, pricing, claims)
  4. If a template is outdated, update it in the repository before reuse

Templates save time and ensure consistency. When Context and Templates conflict, Context wins — update the template to match reality.
